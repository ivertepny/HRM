🔄 Інтеграції
- Slack Bot (через Webhook): подача на відпустку, нагадування.
- Google Calendar API: для синхронізації подій.
- CI/CD (GitHub Actions, Docker): деплой бекенду, фронту, мобілки.


📱 Mobile App (Flutter)
- Авторизація
- Перегляд профілю
- Подача на відпустку
- Відмітка присутності
- Сповіщення (Push)


⚙️ DevOps
- Dockerized для всіх частин (React, Django, Flutter Web).
- CI/CD деплой через GitHub Actions → VPS / AWS.
- Nginx як реверс-проксі для всіх запитів.
- HTTPS через Let's Encrypt.
